# 🚀 Skafferi-Kollen

Skafferi-Kollen is a fullstack web application developed as part of a TE4 exam project.  
The application is designed to help users manage and keep track of food items in their household.

---

## 🎯 Purpose & Target Users

This application is built for:

- Individuals and families who want better control over their pantry
- People who want to reduce food waste
- Users who struggle with keeping track of expiration dates and available ingredients

---

## ❗ Problem

Many people:

- Forget what they already have at home
- Buy duplicate groceries unnecessarily
- Waste food due to poor organization
- Lack a simple digital system for managing their pantry

---

## ✅ Solution

Skafferi-Kollen provides a centralized system where users can:

- Add food items to their pantry
- View all stored items
- Update and edit existing items
- Remove items when they are used or expired

The application ensures that users always have an up-to-date overview of their inventory.

---

## 🧩 Features

- ✅ Full CRUD functionality (Create, Read, Update, Delete)
- ✅ REST API built with Express
- ✅ Persistent database storage
- ✅ Structured backend architecture (routes, controllers, services)
- ✅ Dynamic frontend UI with React
- ✅ Form handling and API integration

### ⚡ Smart Input System

The application supports two different ways of adding products:

- 📝 **Detailed Mode**  
  Users can enter full product information (such as name, quantity, expiration date, category, etc.) for complete control and accuracy.

- ⚡ **Quick Add Mode**  
  Users can quickly add items with minimal input.  
  The system automatically structures and organizes the data efficiently.

This allows:
- Advanced users to manage detailed information  
- Casual users to add items quickly without extra effort  

👉 This feature improves usability, reduces friction, and makes the system flexible for different types of users.

---

## 🏗️ System Architecture

Client ➜ Backend API ➜ Database ➜ Backend ➜ Client

- **Frontend:** React + TypeScript (Vite)
- **Backend:** Node.js + Express
- **Database:** MongoDB / PostgreSQL
